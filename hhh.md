# How to Save From AI

This guide shows how to turn a useful AI conversation into a reusable note.

## What is worth saving?

Save things you will want to reuse:

- stable preferences (tone, format, examples)  
- facts about you or your company  
- recurring tasks and instructions  
- important decisions and constraints  
- prompts that worked well

## Easiest method: let the AI help

At the end of a good conversation, paste this into the chat:

```text
Turn this conversation into a reusable Markdown note.

Keep only the important long-term context, preferences, decisions,
constraints, and reusable instructions.

Use:
- a short title
- a 2–3 sentence summary
- clear section headings
- bullet points

Remove filler and one-off details.
```

Then copy the result into a new `.md` file in the right folder.

## Manual method: use this template

If the conversation is short, you can write it yourself:

```md
# Title

## Summary
2–3 lines about what this note is for.

## Key context
- Main fact 1
- Main fact 2

## Preferences
- Preference 1
- Preference 2

## Instructions
- Reusable instruction 1
- Reusable instruction 2

## Notes
- Extra details if needed
```

Save the file with a clear name, for example:

- `writing-style.md`  
- `company-context.md`  
- `product-roadmap-summary.md`

You can then reuse this note with any AI by copying it or including it in an export bundle.
